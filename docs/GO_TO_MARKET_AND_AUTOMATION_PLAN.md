# Go-to-Market and Automation Plan

이 문서는 "플랫폼 소개 문구"를 실제 사용자 행동으로 연결하기 위한 실행 계획이다.
목표는 아티스트/갤러리가 실제로 들어와서 반복 사용하게 만드는 것이다.

## 1) 플랫폼 소개 문구 (외부용)

ROB는 아티스트의 활동 기록을 신뢰 가능한 프로필로 만들고,  
갤러리의 발굴-검증-연결 비용을 줄여 실제 협업 기회를 높이는 플랫폼입니다.

아티스트는 커리어를 증명 가능한 형태로 축적하고,  
갤러리는 데이터 기반으로 후보를 선별해 아웃리치를 체계적으로 운영합니다.

## 2) 실제로 굴러가려면 필요한 핵심 조건

### A. 사용자 가치가 첫 세션에서 체감되어야 함

- 아티스트 첫 방문 10분 내:
  - 프로필 기본 정보 입력
  - 전시/활동 1건 등록
  - 공유 가능한 프로필 링크 생성
- 갤러리 첫 방문 10분 내:
  - 후보 탐색 1회
  - 후보 상세 확인 1회
  - 아웃리치 큐에 1건 저장

### B. 양측의 핵심 루프가 닫혀야 함

- Artist Loop: 가입 -> 활동 등록 -> 프로필 완성 -> 피드백/기회 수신
- Gallery Loop: 후보 탐색 -> 검증 -> 컨택 -> 응답 추적 -> 재접촉

### C. 운영 지표가 매일 보이고 의사결정에 연결되어야 함

- Acquisition: 신규 가입(아티스트/갤러리)
- Activation: 첫 활동 등록률, 첫 탐색 완료율
- Engagement: 주간 재방문률, 주간 활동 등록 수
- Conversion: 컨택 응답률, 매칭/미팅 전환율

## 3) 실행 방식 (8주 로드맵)

### Phase 1 (1~2주): Activation 완성

- 아티스트 온보딩 3단계 축소
- 전시 등록/타임라인 표시의 핵심 경로 안정화
- 갤러리 탐색 필터 최소 기능 완성

완료 기준:

- 아티스트 첫날 활성화율 40%+
- 갤러리 첫 탐색 완료율 50%+

### Phase 2 (3~5주): Outreach 실험 루프

- 갤러리 리드 수집 품질 기준 수립
- DM/메시지 A-B 템플릿 운영
- 응답률 기반 템플릿 개선

완료 기준:

- 주간 유효 리드 50+
- 7일 응답률 8%+

### Phase 3 (6~8주): 신뢰/확장

- 아티스트 프로필 신뢰 신호 강화 (활동 연속성, 검증 포인트)
- 갤러리 파이프라인 대시보드 고도화
- 주간 리포트 자동화 정착

완료 기준:

- 재방문률/응답률 동반 상승
- 운영팀 수동 리포트 시간 50% 절감

## 4) 에이전트 자동화 가능 범위

### 자동화 가능 (높은 신뢰)

- 백로그 정리: 이슈 생성, 라벨링, 우선순위 큐 관리
- 개발 루프: 코드 변경, 테스트 실행, 실패 원인 요약, PR 초안
- 리포팅: 일일 업무일지, 주간 KPI 요약 문서 생성
- 콘텐츠 초안: SNS 포스트/DM 카피 초안 A-B 생성

### 반자동 권장 (승인 필요)

- 외부 발송: DM/댓글 발송 (수동 승인/실행)
- 정책 민감 수집: 플랫폼 제재 위험이 있는 데이터 수집
- 사용자 데이터 변경: 개인정보 스키마/처리 로직

### 자동화 가능 (조건부 허용)

- 외부 이메일 캠페인 발송:
  - 수신 거부(옵트아웃) 링크 필수
  - 빈도 제한 및 중복 방지 적용
  - 발송/오픈/클릭/반송 로그 저장
  - 클레임/반송 급증 시 자동 중단
  - 구현 엔드포인트: `/api/cron/gallery-outreach-emails` (일일 배치)

### 자동화 비권장 (사람 주도)

- 법적 해석이 필요한 정책 판단
- 대외 메시지 톤 최종 승인 (브랜드 리스크)
- 제재/클레임 발생 시 대응 커뮤니케이션

## 6) 운영 규칙 요약

- 자동화는 "속도"가 아니라 "재현성 + 학습 속도"를 높이기 위해 사용한다.
- 승인 게이트 없는 대외 발송은 금지한다.
- KPI 개선에 기여하지 않는 작업은 보류한다.
