# Goal 03 - Artist Recruitment via SNS Marketing

이 Goal은 SNS를 통한 아티스트 모집을 "콘텐츠 캘린더 -> 제작 -> 발행 -> 성과분석" 루프로 자동화한다.
광고/콘텐츠/DM 실험은 모두 측정 가능한 단위로 운영한다.

## Goal Card

- Goal ID: G-03
- Goal 제목: SNS 기반 아티스트 모집 퍼널 자동화
- 기간: 2026-04-20 ~ 2026-05-04 (2주)
- Owner (CEO): Growth Owner
- Tech Owner (CTO): Tech Lead
- 상태: planned

## Why (배경)

- 모집 채널이 단발성 운영이면 리드 수급이 불안정해진다.
- 콘텐츠-랜딩-전환의 연결 지표가 없으면 개선 우선순위를 잡기 어렵다.

## Success Metrics (KPI 2~3개)

- KPI 1: 주간 모집 리드 수
  - 현재값: TBD
  - 목표값: 주 30명 이상
  - 측정 방법: 폼 제출/상담 신청 기준 집계
- KPI 2: 랜딩 전환율
  - 현재값: TBD
  - 목표값: 5% 이상
  - 측정 방법: 채널별 유입 대비 모집 전환 비율
- KPI 3: 콘텐츠 운영 지속성
  - 현재값: 비정기
  - 목표값: 주 5포스트 이상 + 스토리/숏폼 주 3건 이상
  - 측정 방법: 캘린더 대비 발행 완료율

## Scope

### In Scope (이번에 한다)

- 주간 콘텐츠 캘린더 자동 생성 (채널별 포맷 포함)
- 모집 메시지/CTA A-B 실험
- 채널 성과 리포트 자동 요약

### Out of Scope (이번에 안 한다)

- 브랜드 리뉴얼 전면 개편
- 대규모 유료 광고 집행 자동화
- 근거 없는 대량 메시지 발송

## Execution Plan (Issue 단위)

- Issue A: 채널별 ICP/메시지 맵 정리
  - 목적: 타겟별 메시지 일관성 확보
  - 완료 기준: 채널 x 페르소나 x CTA 매트릭스 문서화
  - 담당: CEO
- Issue B: 콘텐츠 캘린더 생성 루프
  - 목적: 주간 계획-제작-발행 자동 사이클 구축
  - 완료 기준: 1주 분량 캘린더 자동 생성 + 리뷰 프로세스 정의
  - 담당: Engineer
- Issue C: 모집 랜딩/폼 개선 실험
  - 목적: 전환율 개선
  - 완료 기준: 카피/CTA 2개 이상 A-B 실험 설계 및 결과 기록
  - 담당: Engineer
- Issue D: 주간 마케팅 리포트 자동화
  - 목적: 반복 실험 학습 속도 향상
  - 완료 기준: 유입/전환/콘텐츠 성과 1페이지 리포트 발행
  - 담당: CTO

## Review Cadence

- 주간 점검 요일: 수요일
- 중간 중단 조건: 채널 정책 이슈, 전환 급락, 부정 피드백 급증
- Goal 종료 조건: KPI 3개 중 2개 이상 달성 + 다음 퍼널 목표 정의

## Final Check

- [x] 채널별 측정 지표와 실험 루프가 있다.
- [x] 콘텐츠 운영 빈도 목표가 있다.
- [x] 전환 개선 액션이 Issue로 분해되어 있다.
