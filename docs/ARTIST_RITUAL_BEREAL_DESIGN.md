# Artist Ritual — BeReal 구조 설계 (차별화 버전)

**핵심 원칙**: 사람을 바꾸는 게 아니라, **상황을 강제한다**

---

## 1. 본질 정의

| BeReal | Artist Ritual |
|--------|---------------|
| "지금 너 뭐 해?" | **"지금 너는 어떤 상태의 작가냐?"** |
| 일상 포착 | **작가의 보이지 않던 순간을 강제로 드러내는 시스템** |
| 사진 | **작품이 아니라 작가의 상태(state)** |

## 3. 핵심 시스템 5개

### 3.1 Ritual Call (알림 시스템)

| 옵션 | 설명 | 특징 |
|------|------|------|
| **A. 하루 1번** | BeReal형 | 안정적, 참여율 높음 |
| **B. 주 3회 랜덤** | 더 "의식" 느낌 | 부담 ↓ |
| **C. 주제 기반 호출** ⭐ | "지금 작업 중인 것 보여줘"<br>"지금 막힌 부분 찍어"<br>"지금 작업실 상태 보여줘" | **차별점**: BeReal은 무작위, Artist Ritual은 의미 있는 질문 |

### 3.2 Dual Capture (작가 버전 카메라)

| BeReal | Artist Ritual |
|--------|---------------|
| 앞+뒤 카메라 | **카메라 1**: 작업물/화면<br>**카메라 2**: 작가 얼굴 or 손 |
| 결과 | **"작품 + 상태" 동시 기록** → 강한 아카이브 |

### 3.3 조건부 열람 구조 (매우 중요)

- **❌ 안 올리면 남 못 봄**
- **⭕ 올려야 타인 작업 열람 가능**

→ 초기 DAU, 참여 강제, 네트워크 효과

### 3.4 Reaction 시스템

- **댓글 길게 쓰게 하면 망함**

| 구분 | 내용 |
|------|------|
| Quick Reaction | 🔥 좋다 / 🤯 충격 / 👀 흥미 / 🧠 구조 좋음 |
| Real Reaction ⭐ | 자기 얼굴로 반응, 또는 짧은 음성 (3초) |
| 이유 | 작가들: 글보다 직관 반응이 빠름 |

### 3.5 Ritual Miss / Late 시스템

| 상태 | 표시 | 기준 |
|------|------|------|
| Active Artist | 🔥 | 0:00~21:30 (콜 전 또는 콜 후 30분 이내) |
| Late Artist | 🕒 | 21:31~자정 |
| Missed Ritual | ❌ | 당일 미기록 |

## 4. Artist Ritual만의 차별화 5가지

### 4.1 "완성작 금지" 규칙

- **완성된 작품 ❌**
- **과정만 허용 ⭕**

→ 인스타화 방지, 진짜 리얼 확보

### 4.2 상태 태그 시스템

사진 + 태그 예시:

- `stuck` / `exploring` / `refining` / `destroying` / `restarting`

→ 쌓이면 **"작가의 사고 패턴 데이터"**

### 4.3 자기 아카이브 자동 생성 (ROB 연결)

- Ritual 기록 → 자동 작업 타임라인
- 전시 준비 과정, 작가 활동 증명
- **예술인 증명 / 포트폴리오 자동화**

### 4.4 크리틱 모드 (선택 기능)

- 특정 Ritual에 "피드백 요청" 체크
- 다른 작가가 짧은 코멘트
- → 커뮤니티 깊이

### 4.5 "같은 순간, 다른 작가" 비교 구조

- 같은 시간에 여러 작가 상태를 한 화면
- → **"작가 집단의 현재 상태"** 가시화

## 5. 작가용 알림 시스템 5가지 컨셉

| 컨셉 | 문구 예시 | 비고 |
|------|-----------|------|
| **1. 작업 호출** | "지금 작업 중인 것 보여줘" | 기본형 |
| **2. 막힘 공유** | "지금 막힌 부분 찍어" | 진정성 |
| **3. 공간 기록** | "지금 작업실 상태 보여줘" | 아카이브 |
| **4. 상태 질문** | "지금 어떤 상태의 작가야?" | 메타 |
| **5. 순간 포착** | "2분 안에 지금 순간 찍어" | BeReal형 |

## 6. Week 1 우선순위 (현실 버전)

### MUST (무조건)

- [ ] Ritual 알림 (하루 1회)
- [ ] 사진 업로드 (간단 버전)

### SHOULD (가능하면)

- ❌ 팔로우/추천 알고리즘

- ❌ 이미지 편집 기능
