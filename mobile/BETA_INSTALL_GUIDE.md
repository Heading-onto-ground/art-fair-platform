# Artist Ritual — 베타 설치 가이드

아티스트 테스터를 위한 설치 방법입니다.

> 📖 **전체 사용 가이드**: [PRIVATE_BETA_GUIDE.md](./PRIVATE_BETA_GUIDE.md) — 테스트 목표, 사용 방법, 해주면 좋은 것

---

## Android 사용자

1. **다운로드 링크**를 받습니다 (빌드 완료 후 전달)
2. 링크를 휴대폰 브라우저에서 엽니다
3. **APK 다운로드** 버튼을 누릅니다
4. 다운로드 후 **설치**를 진행합니다
   - "알 수 없는 출처" 허용이 필요할 수 있습니다
5. **Artist Ritual** 앱을 실행합니다

---

## iPhone 사용자

1. **TestFlight** 앱을 App Store에서 설치합니다 (없다면)
2. 초대 **이메일** 또는 **TestFlight 링크**를 받습니다
3. 링크를 탭하면 TestFlight가 열립니다
4. **설치** 버튼을 누릅니다
5. 설치 후 **Artist Ritual** 앱을 실행합니다

---

## 앱 사용 방법

1. **Enter** → 로그인 (아무 이메일 + 비밀번호로 테스트 가능)
2. **Record Moment** → 사진 촬영/업로드 → Note, State, Medium 입력 → Save
3. **Feed** → 다른 작가들의 작업 순간 피드
4. **Profile** → 내 활동 기록, 알림 설정
